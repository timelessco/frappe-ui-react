# Button

A versatile button component that supports different sizes, variants, states and content types including icons and loading spinners.

<Image src={image} alt="Button" className="w-full rounded-lg shadow-lg" />

## Features

- Multiple sizes: sm, md, lg, xl, 2xl
- Different variants: solid, outline, ghost, subtle
- Support for prefix/suffix icons
- Loading state with customizable spinner
- Icon-only mode
- Haptic feedback support
- Scale animation on press
- Accessibility support
